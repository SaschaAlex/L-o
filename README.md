This is an interpreter for the Foobymbol language 


## Requirements
* A working version of python 3.6+

## Installation 


Clone :
```sh
git clone https://github.com/SaschaAlex/L-o
python interfaceC.py
```

### Usage example
```
In[0] (4/3x + 31 = 19/3 + 97x)
Out[0] x = 74/287
```
